How to get Support

This part of the course requires a lot of self-directed learning to take place. You must be proactive and willing to help yourself. Of course there is support available in a number of ways. Please check for the answer to your question using the resources available in the order below.

  1. Frequently asked questions: There is a FAQ section for commonly occurring errors, which will be added to as and when we come across more during the sessions!
  2. Video Resources: A video will accompany each task in the activity sets
  3. Cribsheets: There is a series of How To cribsheets to assist with commonly completed tasks, these can be found on the How To... page
  4. EDP Forum: There is a PsychoPy Builder forum on the Moodle. This is the best way to get help outside class and if you ask a tutor directly, they will refer you to the EDP Forum
