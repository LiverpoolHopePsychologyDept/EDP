How to Crib Sheets

We have supplied crib sheets to show you how to use PsychoPy Builder and the word/nonword databases for your assessment. Some of the crib sheets go above and beyond the scope of the assessment, but are available for you if you want to explore PsychoPy Builder further.

  1. Monitor Settings
  2. Experiment Settings
  3. Naming Conventions
  4. Using Conditions Files
  5. Apply Participant Feedback
  6. Using the MRC Psycholinguistic Database
  7. Using the ARC Nonword Database
  8. Using the wordNorms.com Database (word pairs)
