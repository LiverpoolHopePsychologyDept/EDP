Submission Guidance

Part 1: Method Section

  1. Log in to Introduction to Psychology (Core 2) Moodle
  2. From this link or the Coursework Coversheets Folder download and complete C2_EDP_Method_Coversheet.docx
  3. Ensure your coursework submission is named correctly, as specified in the coversheet document
  4. Click Turnitin Assignment 2 EDP Part 1: Method Section
  5. Submit ONLY the coversheet and method section for this part of the assessment

Part 2: PsychoPy Program

  1. From this link or the Coursework Coversheets Folder download and complete C2_EDP_Program_Coversheet.docx
  2. Click Assignment EDP Part 2: Program Submission
  3. Click Add Submission and a drag and drop box/select files box will appear
  4. If you have used folders in your program, then you will need to recreate them in the File Submissions box
  5. Upload your PsychoPy Program and any associated/necessary files such as conditions files and ensure your files are in the correct folders (if used)
  6. You should also submit your Method Section here.
  7. Click Save changes

Do not upload the data folder/files, this will exceed the maximum file upload limit
